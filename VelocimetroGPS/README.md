
# Pomodoro

Sencillo timer para aplicar la técnica de estudio Pomodoro

## Funciones

- Tiempo de estudio fijo a 25 minutos
- Tiempo de descanso fijo a 5 minutos
- El botón A permite pausar el tiempo de estudio o descanso
